Pebble Vina × Pebble Square

Bản trình chiếu đã chuyển sang ./deck

File index.html này (bản demo nhanh cũ) đã nghỉ hưu để tránh hai bản render đua nhau. Tầng trình chiếu chính thức giờ là app Next.js tam ngữ trong thư mục deck/.

Chạy:

cd deck && npm install && npm run dev → http://localhost:3000

Điều khiển: ← → chuyển · l đổi ngôn ngữ (VI/KO/EN) · m bảng meta · a phụ lục · f toàn màn hình
